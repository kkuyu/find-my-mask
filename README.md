# Find My Mask

> 마스크 품목명, 업체명, 등급, 허가일자, 허가상태, 등 상세 정보 확인 서비스입니다.

![](/static/share-image.png)

## 개발 환경

```bash
npm >= 7.10.0
node >= 16.0.0
```

## 설치 방법

```bash
npm install
npm run dev
```

## 브라우저 지원

- Google Chrome
- Safari
- Firefox
- Edge

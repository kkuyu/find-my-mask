# Find My Mask

> 공공데이터포털 오픈 API를 이용한 보건용·수술용·비말차단용 마스크 허가 현황 확인 사이트

## API

| Name | Provider | Description |
| ----- | ----- | ----- |
| 식품의약품안전평가원\_의약외품 제품 허가정보 | 식품의약품안전처 식품의약품안전평가원 | 의약외품의 품목명, 용법용량, 효능효과, 사용상의 주의사항 등 정보를 제공하는 공공데이터 |
| 의약품 회수·판매중지 서비스 | 식품의약품안전처 | 식품의약품안전처에서 관리하는 의약품(허가,행정,성분 등)의 관련 정보 조회 |
